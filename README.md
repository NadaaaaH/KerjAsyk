# KerjAsyk 🚀

**KerjAsyk** adalah platform kolaboratif dengan pengalaman visual dinamis (parallax) yang dibangun untuk mendukung produktivitas tim.

## Fitur
- **Parallax Scenes:** Visual interaktif yang imersif.
- **Modern UI Components:** Antarmuka responsif menggunakan komponen standar industri.
- **Optimized Performance:** Dibangun dengan tool development generasi terbaru.

## Tech Stack
Proyek ini dikembangkan menggunakan teknologi berikut:
- **Framework:** [React 18](https://reactjs.org/) dengan **Vite**
- **Language:** [TypeScript](https://www.typescriptlang.org/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **UI Library:** [shadcn/ui](https://ui.shadcn.com/) (Radix UI & Lucide Icons)
- **Runtime:** [Bun](https://bun.sh/) / [Node.js](https://nodejs.org/)
